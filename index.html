Beautify

An iOS app written entirely in Swift, which let users design their own interface. So basically, what this app does is that it lets a user play with different color combinations by dynamically modifying RGB values through a wheel. User can change the color of the font too, by triggering the switch to lock the background. In short, a user can make the interface of his own choice

Features

  • Flat background coloring variations
  • Text color variations
  • Taste of all the available iOS fonts
  • RGB representations of each variation
  • Dynamic variations in the Font Family
  • Use of the new Blur View introduced in iOS8
  • Addition of UIDynamics introduced in iOS7
  • Variations in the Font Style
  • Default templates through the side bar
  • Material Design Components
  • Twitter integration
  • Final animated result when Done

Compiled on Xcode 6.1, Optimised both for iPhones and iPads running iOS 7 or later.
